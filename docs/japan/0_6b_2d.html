　 不思議な門に取り込まれた二人は
痛いほどの静寂中…ゆっくりと目を開く
　 そこには光に満ちた
何もない世界が広がっていた

ロギ		 …う…ここは？
　 背後に巨大な存在を感じ取り、
二人はハッと振り返る

ディオス		 な、何だ、これは…!?
？？？ 《真理の扉》…魂に審判を下す場所

ディオス		 何者だ!?
ウロボロス 我はウロボロス…真理と世界を繋ぐ、
運命の見届け人
ウロボロス …うぬらに『魂の繋がり』を見た
うぬらには《力》を持つ資格がある

ロギ		 《力》を持つ資格…？
ま、待ってくれ、これはいったい!?
ウロボロス 世が揺らぎ、未来の見えぬ今…
《力》はうぬらに平穏を与えない
ウロボロス 運命は荒れ狂った激流となり、
あまたの人々を飲み込むであろう
ウロボロス 《力》無くして、人類に平穏は訪れん
だが…《力》の意味を知らぬものに、
真の《力》が与えられることはない

ディオス		 力…？
何なんだ、その力とは…？
ウロボロス うぬらに問おう!!
《力》が欲しいか？
…すべてはうぬらの《意志》に委ねられた!!

ロギ		 僕は、
《力》が欲しい…
この愛しき世界を守る《力》が…！

ディオス		 俺は、
《力》が欲しい…
この悲しき世界を変える《力》が…！
ウロボロス …よかろう…今ここに、
真理の契約は結ばれた！
　 ウロボロスが叫ぶと
二人の体を淡い光が包み込んだ…
ウロボロス うぬらの《意志》が目指す未来…
その先に、世界はどのような審判を下すのか
答えは扉の向こう側にある
ウロボロス …あがけ人間どもよ！
そして、揺るぎなき真理を導いてみせよ！
